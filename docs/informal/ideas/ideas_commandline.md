* ls/dir: responsive object in sight
* pwd: location of the hero
* whoami: hero attribute page
* cd: teleport to given location
* top: task list
* mkdir: new house
* mv: move house to a give location
* rm: remove house
* date: time in game world
* history: used command

##Second way of calling the command line
* :w: save
* :q: quit(warning if unsaved)
* :q!: quit without saving
* ZZ/:wq: save&quit
